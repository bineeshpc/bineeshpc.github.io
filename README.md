## 🚀 Getting Started

### Prerequisites
- Python 3.x installed on your machine

### Running Locally

1. Clone the repository:
```bash
git clone https://github.com/bineeshpc/bineeshpc.github.io.git
cd bineeshpc.github.io
```

2. Start a local server:
```bash
python3 -m http.server 8000
```

3. Open your browser and navigate to:
```
http://localhost:8000
```
